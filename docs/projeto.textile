h1. DCOO 2013/02 - Projeto Tênis - Exemplo Git

h2. Enunciado

Implemente a classe Tênis (o jogo ? ). Para criar um jogo de tenis percisamos ter no mínimo 2 jogadores e no máximo 4 jogadores. Cada jogo de tenis, além de ter os jogadores, também precisa ter um placar. Modele os seus jogadores como classes Pessoa a qual tenha nome e idade, use-a para intanciar um jogo de tênis. Implemente os métodos necessários para imprimir o placar do jogo, mudar o placar, definir o vencedor e perdedor. Implemente uma classe main responsavel por isntanciar testar seu objeto.

* *T001* - Criação da infra do projeto
* *T002* - Planejamento das tarefas
